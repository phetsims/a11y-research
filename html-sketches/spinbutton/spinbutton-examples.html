Spinbutton Examples

Using div element

The div example is valid HTML.

Mass 1
<div> class="spinbutton form-element" tabindex="0" role="spinbutton"
aria-valuemin="1" aria-valuemax="10" aria-valuenow="4" 
aria-valuetext="billion kilograms">Mass 1</div>

Using button element

Putting role=spinbutton and using the aria-min,-max,-now, and -text attributes is not valid HTML.

<button> role="spinbutton"
aria-valuemin="1" aria-valuemax="10" aria-valuenow="4" 
aria-valuetext="billion kilograms">Mass 1</button>
