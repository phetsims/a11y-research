Custom Scene Graph - Scenery

  • Open source sim architecture (by PhET) that runs all the HTML5 simulations
  • Data tree structure that represents each graphical element within parent and child nodes
  • Handles the logical & spatial arrangements of the graphical scene
  • For performance & flexibility Scenery uses multiple graphics renderers SVG, Canvas, and WebGL

Making these important learning tools accessible to more students will have a big impact!

Accessibility Challenges

Scenery's Challenges

  • Provides no semantic structure
  • Employes multiple graphics renderers
  • Everything is drawn on the screen, even text

Design Challenges

  • Sims are highly interactive
  • Multiple changes can occur with a single interaction
  • Wide range of interactive sim objects

Need a robust accessible solution, agnostic to graphical renderer!

Innovative Parallel DOM

PDOM = HTML Tree

  • Data nodes rendered as HTML elements
  • Leverage structural & interaction semantics of HTML & ARIA
  • Creates a powerful & flexible accessibility layer
  • Robust features agnostic to the graphics renderer

Design Benefits

  • Keeps all accessibility information in one place
  • Minimizes long term maintenance costs
  • Reduces complexity of sim code
