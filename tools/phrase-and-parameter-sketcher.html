Phrase and Parameter Sketcher

The Idea

Iâ€™m wondering if a wizarding tool could be created that allowed a designer to build and play with a phrase and any number of parameters within the phrase?

The tool might allow the designer to:

  1. Create a base phrase
  2. Identify any number of parameter-blocks for phrase.
  3. Name the parameter-blocks
  4. Place named parameter-blocks within the phrase at any location.
  5. Populate each parameter-block with a list of parameters (min 2).
  6. Play with the phrase and the created parameters
  7. Export the phrase and the list of parameters into somethign useful (maybe some markdown) that could be embedded in design doc (or into json code)?

Step 1: Choose a sim to load in to the iframe

Not required, but it might be nice to be able to embed the sim into the tool.

Step 2: Choose a structure for the description

Ideally, a designer could choose between a few simple structures in which to house the description (e.g. a paragraph, or a list, option to add a heading)

Step 3: Build the parameterized description

Based on the structure chosen in Step 2, build the parameterized description in a paragraph or list.
